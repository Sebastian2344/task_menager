# task_meneger

Aplikacja do zarządzania zadaniami.
### Funkcje:
  * tworzenie,usuwanie,edytowanie zadań,
  * wyświetlanie zadań wykonanych i do wykonania,
  * informacje o najbardziej produktywnym dniu w tygodniu,
  * wyświetlanie ilości zadań wykonanych i do wykonania,
  * notyfikacje lokalne.
### Testy:
  * jednostkowe,
  * widget testy.
### Biblioteki:
  `sqflite`,
  `provider`,
  `path`,
  `flutter_local_notifications`,
  `permission_handler`,
  `timezone`,
  `mocktail`,
  `flutter_test`.
